World Wide Web pioneer calls for AI to preserve 'original values' of web

Japan Today Tech · 2026-06-05
ワールドワイドウェブの生みの親ティム・バーナーズ＝リーは、AIがウェブの「本来の価値」を守りつつ、個人データをテック大手に送る前にユーザーが選別できる仕組みが必要だと訴えている。彼はAIにはW3Cのような標準化の枠組みがなく、個人データの管理が急務だと指摘し、スタートアップInruptのAIアシスタント「Charlie」でその実現を目指している。

Grammar focus

End-position reporting clauses

Place 'he said/told/warned/added' at the end of a sentence after a quote or reported content.

"Charlie is about preserving the original values of the web," he added.

英語ニュースでは発言内容を先に述べ、文末に he said / he told AFP / he added などを置いて話者を示す構文が非常に多い。日本語の「〜と述べた」に相当する。

benefit from + gerund

Use 'benefit from' followed by a gerund (-ing) to show gaining advantage from an action.

AI would benefit from being regulated.

benefit from の後ろは動名詞を取る。「〜することから利益を得る」。being regulated のように受動の動名詞も可能。

Participial phrases for background

Use past participles or -ing phrases at the start of a sentence to give background, reason, or simultaneous context.

Determined to make the web freely available, he did not patent his program.

文頭の分詞構文（Determined to ... / Launched in 2018 / With ... preoccupying ...）で背景・理由・同時状況を簡潔に追加する。ニュース記事で頻出する高度な構文。

With + object + present participle

Use 'with + noun + -ing' to describe a situation where something is happening.

With the use of personal data preoccupying authorities, Berners-Lee has made data protection his main cause.

with + 名詞 + 現在分詞 で「〜が…している状況で」という複雑な背景を一文で表す。原因や文脈設定に使われる。
