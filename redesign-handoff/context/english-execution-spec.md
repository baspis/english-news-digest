- Top-level goal: use English fluently and independently as a practical tool.

This file exists to prevent the plan from becoming vague, over-rich, or test-centered.

## Non-negotiable principle

Do not optimize for a rich plan.

Optimize for:

1. avoiding critical learning errors,
2. preserving the four required learning functions,
3. keeping the loop executable for years,
4. making English usable in real life,
5. measuring progress honestly,
6. protecting health, sleep, family, work, and SAD stability.

## Fixed architecture

### Four learning pillars

| Pillar | Required function | What belongs here | What does not belong here |
| --- | --- | --- | --- |
| Input | Understand English and notice usable language | Listening, reading, re-listening, real-world materials, perception training, phrase noticing | Passive consumption with no reuse for weeks |
| Output | Create meaning in English | Speaking, writing, role-play, practical tasks, clarification, paraphrase, communication repair | Silent study, reading corrections only |
| Language Focus | Explicitly improve form | Grammar, vocabulary, chunks, pronunciation, prosody, collocation, pragmatics, register, error correction | Long test-taking sessions, passive media consumption |
| Fluency | Use known English faster and with less effort | Repeated speech, timed writing, speed reading, re-answering, re-listening, automaticity drills | New grammar study, dictionary browsing, tool exploration |

### Two control layers

| Layer | Role | Examples |
| --- | --- | --- |
| Feedback / Measurement | Prevent wrong practice from continuing | AI correction, re-output, recordings, writing samples, external checks, 90-day audit |
| Sustainability / Self-regulation | Keep the system alive for years | minimum week, winter/SAD mode, sleep protection, no catch-up punishment, health-first scheduling |

Rule:

- Never create a fifth learning pillar unless the four-pillar model clearly fails.
- New concerns must first be mapped into one of the four pillars or two control layers.
- If an activity cannot be mapped, it is not part of the plan.
