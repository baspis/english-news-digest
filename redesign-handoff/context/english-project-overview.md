## Core conclusion

The top-level goal is not IELTS.

The goal is to use English fluently and independently as a practical tool for:

- life,
- work,
- travel,
- reading,
- listening,
- information gathering.

IELTS Overall 8.0 remains a long-term measurement target and calibration instrument. It is not the purpose.

## Fixed architecture

Use four learning pillars:

| Pillar | Function |
| --- | --- |
| Input | Understand English and notice reusable language |
| Output | Create meaning in English through speaking and writing |
| Language Focus | Improve grammar, vocabulary, chunks, pronunciation, pragmatics, and register |
| Fluency | Use known English faster and with less effort |

Use two control layers:

| Control layer | Function |
| --- | --- |
| Feedback / Measurement | Correction, re-output, recordings, writing samples, external calibration |
| Sustainability / Self-regulation | SAD-aware design, health priority, sleep protection, minimum week, failure recovery |

Do not create new pillars. New tools, concerns, or anxieties must be mapped into this structure first.
